ProjektGaleria - projekt mobilnej aplikacji do otwierania zdjęć.
Aplikacja przeznaczona jest na urządzenia z systemem Android. Działa na systemach Android 5.0(Lolipop) lub nowszych.

Projekt można uruchomić w programie Android Studio.
Wersja: Android Studio 11.0.10+0-b96-7249189 amd
VM: OpenJDK 64-Bit Server VM by Oracle Corporation
